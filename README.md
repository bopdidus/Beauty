# BeautyMessageBox
BeautyMessageBox works exactly like default MessageBox of .NET 

## BeautyMessageType
- Error
- Warning
- Primary
- Secondary
- Success

## BeautyMessageIcon
  - TickMark
  - Question
  - CrossMark
  - Information
  - ExclamationMark

## BeautyMessageButtons
  - YesNo
  - OK
  - CancelTryContinue
  - OKCancel
